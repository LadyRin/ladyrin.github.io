Ars Eternalis

Présentation

Ars Eternalis est un prototype de FPS Roguelike réalisé sur Unity. Il a été réalisé dans le cadre de la 7e édition de la Code Game Jam au cours de 30 heures.

Préparation

Avant même l'annonce du thème, j'ai personnellement formé mon équipe à Unity et C#, de sorte que nous puissions nous concentrer sur le développement du jeu une fois le thème annoncé.

Game Design

Le thème de la Code Game Jam 2022 était "Maître du temps". Nous avons donc décidé de créer un jeu dans lequel le joueur disposait de pouvoirs temporels. Le joueur devait donc utiliser ces pouvoirs pour survivre dans un donjon généré procéduralement.

Malheureusement, nous n'avons pas eu le temps de développer toutes les fonctionnalités que nous avions prévues. Au final, la gameplay loop du jeu n'était pas finie. Nous avons cependant réussi à implémenter les fonctionnalités suivantes :

  • Base FPS du jeu (déplacement, saut, tir, etc.)
  • Génération procédurale du donjon
  • Ennemis
  • Système de pouvoirs temporels

Réalisation

Le jeu a été réalisé en 30 heures. Il a été réalisé en utilisant les langages et technologies suivants :

  • Unity
  • C#
