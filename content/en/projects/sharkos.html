SharkOS

SharkOS is my digital portfolio and the website you are currently visiting. It is a web application built with VueJS that resembles an operating system. It showcases my projects, skills, and professional experiences. You can also contact me through the website.

This site is primarily a personal project, created to showcase my web development skills and highlight my projects. As a (fake) operating system, it offers the following features:

  • A terminal for executing commands
  • A file system used to store web pages presenting my projects
  • A window manager for visualizing the projects
  • A desktop with shortcuts to different sections of the site
  • A "Dolphin" application for file navigation
  • A "Firefox" application for dynamically loading either HTML pages or VueJS components
