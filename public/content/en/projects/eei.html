Exactly Enough Items

Exactly Enough Items (EEI), named after the Minecraft mod "Just Enough Items", is a web application that I created as part of a course project in my third year of studies. It is a web application that allows managing items and recipes for the Minecraft game. The application is divided into two Docker containers, one for the front-end using VueJS, and one for the back-end using Symfony, API Platform, and PostgreSQL. A secondary Symfony application, MyAvatar, is used to manage user profile pictures using the MD5 hash of their email address.

The main purpose of the application is to allow a user to view the game's items and the recipes to craft them. To achieve this, an administrator can add items, recipes, and tags for the items. Users can then search for items by name, view the recipes to craft them, and see the items that have a certain tag. Recipes can also make use of tags, for example: a chest can be crafted with any type of wood. A premium user (premium accounts can be purchased through the Stripe API) can also create their own items, recipes, and tags. These will then be hidden from other users.

This site was developed using Docker, VueJS, Symfony, API Platform, and PostgreSQL. It was created as a group project with four other students. The source code will soon be available on GitHub.
