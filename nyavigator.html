Nyavigator

Présentation

Nyavigator est un site web de navigation routière. Il permet à ses utilisateurs de créer des itinéraires entre des communes françaises, et de les enregistrer comme favoris.

L'objectif du projet était l'optimisation d'une application existante. Le projet a été réalisé en équipe de 4 personnes, et s'est déroulé sur 8 semaines.

Améliorations apportées

L'application a été améliorée en plusieurs points :

  • Optimisations de la base de données: La façon dont les données étaient stockées a été revue afin de réduire le nombre de requêtes nécessaires pour récupérer les voisins d'un noeud routier. L'utilisation d'index, de vues matérialisées, et de meilleures requêtes SQL a permis de réduire le temps de réponse de l'application.
  • Optimisations de l'algorithme: L'algorithme de recherche de plus court chemin a été optimisé afin de réduire le temps de calcul. Originellement basé sur l'algorithme de Dijkstra, il a été modifié pour utiliser une structure de données plus adaptée (une file de priorité) et a été remplacé par A* pour réduire le nombre de noeuds explorés.
  • Amélioration de l'interface: L'interface a été revue afin de rendre l'application plus intuitive et plus agréable à utiliser. L'ajout d'une carte interactive grâce à Leaflet et OpenStreetMap a permis de visualiser les itinéraires.
  • Ajout d'une API REST: Une API REST a été ajoutée afin de permettre l'asynchronisme entre le client et le serveur. Elle permet de récupérer les données nécessaires à l'affichage des itinéraires (coordonnées géographiques, entre autres), d'enregistrer des itinéraires comme favoris, et de récupérer les itinéraires favoris d'un utilisateur.

Réalisation

Nyavigator a été réalisé en utilisant les langages et technologies suivants :

  • HTML
  • SCSS
  • JavaScript
  • PHP
  • Composants Symfony
  • PostgreSQL
  • Git

Le projet à été réalisé dans le cadre d'une SAÉ (Situation d'Apprentissage et d'Évaluation) de 2ème année de BUT Informatique.
